## **ğŸ› ï¸ The Tool: Developer Extension**

The developer extension is your AI coding companion \- it's **enabled by default** in goose and gives you powerful capabilities:

**Key Tools:**

* **text_editor**: Create, view, and edit files (supports view, write, str\_replace, insert, undo\_edit)  
* **shell**: Execute commands in your terminal  
* **analyze**: Understand code structure and relationships  
* **screen_capture**: Take screenshots for visual debugging  
* **image_processor**: Process and work with images

**For this challenge**, you'll primarily see goose use **text\_editor** to create your files. Watch in the UI as goose uses these tools \- it's completely transparent about what it's doing\!

**Three approaches you can take:**

1. **One-shot prompt**: Describe everything you want in one detailed prompt  
2. **Iterative**: Start simple, then refine through conversation (recommended for learning\!)  
3. **Plan:** Have goose create a plan before executing\!
