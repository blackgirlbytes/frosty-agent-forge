## Getting Started

Use goose with multiple extensions:

```bash
# Make sure you have the deployment extension installed
goose extension add vercel
# or
goose extension add netlify

# Start building
goose session start "Build and deploy a winter festival website"
```

Then describe what you want and let goose orchestrate the extensions!
