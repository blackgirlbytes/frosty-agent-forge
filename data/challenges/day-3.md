## The Crisis

The annual Hot Cocoa Championship just concluded with 12 contestants, 200 votes, and a mountain of data. Unfortunately, the festival's data analyst called in sick, and the awards ceremony starts in two hours! The festival director needs beautiful visualizations to display on the big screen.

## Your Mission

Transform raw tournament data into stunning interactive visualizations. You'll need to:

- Create a tournament bracket visualization
- Show vote distributions across contestants
- Compare recipe ingredients and techniques
- Make it interactive and presentation-ready
- Ensure it looks great on a projector

## What You'll Learn

- **Data Visualization**: Turn unstructured data into meaningful charts
- **MCP Extensions**: Use the auto-visualiser extension
- **MCP-UI**: Create interactive visualizations that work across platforms
- **Data Transformation**: Clean and structure messy data

## Getting Started

Install and use the auto-visualiser extension:

```bash
# Install the extension
goose extension add auto-visualiser

# Start your session
goose session start "Visualize hot cocoa championship data"
```

Then provide your data and ask goose to create visualizations!
